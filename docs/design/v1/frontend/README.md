# RepoPilot v1 前端（设计归档）

> **已迁入主线：** 正式开发与构建请使用 [`apps/web/`](../../../apps/web/)。本目录保留设计规格、审查记录与 HTML 原型，供对照参考。

React + Vite + TypeScript 实现，数据来自 `MockApiClient`（`VITE_USE_MOCK=true`）。

## 快速开始（归档副本）

```bash
cd docs/design/v1/frontend
npm install --include=dev
npm run dev
```

主线开发：

```bash
# 仓库根
npm install
npm run dev:web
```
